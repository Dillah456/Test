Keterangan Mekanik

Dzikr

  • Empat penghitung: Hamdalah, Tasbih, Takbir, Istigfar.
  • Setiap penghitung ditambah dengan menekan tombol +1 masing-masing.
  • Setiap 33 total Hamdalah + Tasbih + Takbir:
    • Menambah +1 Ijtihad
    • Menambah +33 MP
  • Istigfar:
    • Setiap 100 poin → menambah +50 Ijtihad

Ijtihad & MP

  • Ijtihad = poin usaha yang dikumpulkan dari dzikr dan Istigfar.
  • MP = diturunkan dari Ijtihad:
    • Setiap +1 Ijtihad → +33 MP

Ekspor Data

  • Tombol Unduh JSON menyimpan semua nilai dzikr, Ijtihad, dan MP ke file .json.
  • Data juga bisa disimpan ke cloud (Firebase) dengan Simpan, dimuat kembali dengan Muat, atau dihapus dengan Hapus.

Ringkasan Progres

  • Tambah dzikr → kumpulkan Ijtihad & MP
  • Akumulasi Istigfar → tingkatkan Ijtihad
  • Unduh JSON kapan saja untuk menyimpan rekap
