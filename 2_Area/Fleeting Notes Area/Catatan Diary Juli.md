Perihal Layering :
Layering ini adalah Konklusi dari diskusi dengan Ai, perihal konsep diri, yang tentunya kompleks dan sulit di Abstraksi, bila pun di Abstraksi maka perlu menjadi Modul di berbagai layer berbeda.

Layer Sistem :
1. Layer Program 
2. Layer Script
3. Aggregated Layer -> Merupakan area dengan Database Driven dan Komunikasi antar Program
4. Layer Markdown
5. Layer Multimedia
6. Presentation Layer ->Merupakan area Data yang dapat diinteraksi langsung 
7. Terminator Layer -> Kumpulan Human Unit yang dapat di Modulasi sebagai Persamaan tapi tidak dapat berinteraksi langsung dalam Sistem

Berbeda dengan Modulasi yang merupakan proses Abstraksi Objek Kompleks, Layering adalah proses penerimaan Kompleksitas, Ketidakpastian, ketidaktahuan dan membatasi Modulasi pada nilai Objektifnya tersendiri, karena itu Tidak ada Entitas yang betul betul muncul di fase Layering.
Tapi sebagai "Presentation Layer".
